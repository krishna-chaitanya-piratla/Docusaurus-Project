# Lists

## Unordered Lists

To create an unordered list, use a dash (`-`), plus (`+`), or asterisk (`*`) followed by a space.

```
- Item 1
- Item 2
- Item 3
```

## Ordered Lists

To create an ordered list, use numbers followed by a period and a space.

```
1. First Item
2. Second Item
3. Third Item
```

## Nested Lists

You can also create nested lists by indenting the items with four spaces or one tab.

```
- Item 1
  - Subitem 1
  - Subitem 2
- Item 2
  1. Subitem 1
  2. Subitem 2
```
